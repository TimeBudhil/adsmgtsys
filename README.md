# ADS Management System

## Requirements

Before running this application, ensure the following software is installed:

* **Java Development Kit (JDK) 25**
* **Apache Maven** (recommended version 3.9 or newer)

Verify your installation:

```bash
java --version
mvn --version
```

## Build the Application

From the project root directory, run:

```bash
mvn clean package
```

This will compile the project and create an executable JAR file in the `target` directory.

## Run the Application

After building the project, run:

```bash
java -jar target/adsmgtsys.jar
```

The application starts using the main class:

```text
com.cs425.adsmgtsys.ADSMgmtSysMain
```
